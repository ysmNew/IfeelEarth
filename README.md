## Aiffle Hackathon

---

- 미니해커톤은 해커톤3를 진행하기 위한 사전 준비
- 최종 결과물로는 코드가 포함된 보고서(양식은 자유)
- 3일간 진행한 내용이 납득할 수 있도록 정성적인 결과물이 나와야함
- 적합한 예시 : 데이터 수집 후 EDA 진행, 보유한 데이터로 베이스 모델에 학습 및 여러가지 모델과 비교 등
- 부적합한 예시 : 데이터 수집만 X, 논문 리서치만 X 등
